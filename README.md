# Pretest-Kollaborationsnetzerk
Ziel des Projekts ist es, die Kollaborationsstrukturen der deutschsprachigen Charts zu analysieren. Mithilfe der Netzwerkanalyse untersuchen wir, welche Rollen Labels, Songwriter und Produzenten in erfolgreichen Songs einnehmen und ob sich zentrale Akteure identifizieren lassen, die wiederholt an erfolgreichen Produktionen beteiligt sind.
